What Should I Drink?

What Should I Drink? is an application created using C#, .Net Core, MVC, jQuery and identity framework. The application allows a user to find a beer to drink based on the time of year, how they're feeling and what type of night the user is seeking.

Biggest challenge: My biggest challenge with this project was chaining the values from the dropdowns into one call to my database. It took sometime to figure out the exact syntax that I needed to get the desired result. I was lucky in that I was looking for three values because it helped me deduce exactly how I needed to set the parameters, including the name of the controller and the method.

Before You GO

An application for Pokemon Go users to plan and track their favorite routes for catching Pokemon, built using AngularJS, Firebase and Materialize. The application also uses the Google Maps Distance Matrix API, Weather Underground API, Poke API and authentication.

Biggest challenge: My biggest challenge with this project was working with the APIs and getting to the data I needed. The Google Distance Matrix API's data was several layers deep and it took some maneuvering to get exactly what I wanted.

Bangazon!

The initial Bangazon site is an application that simulates an online retailer like Amazon. In the application, users are able to register as a customer, add products and simulate purchasing products listed by other customers. The application uses the .Net framework, MVC pattern and SQLite.

Biggest challenge: My biggest challenge with this project was implementing the singleton pattern across all the various pages, tying the active customer across the entire application.
